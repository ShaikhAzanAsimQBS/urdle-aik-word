# اُردل — Urdu Wordle

A mobile-first Urdu Wordle game built with pure HTML, CSS, and JavaScript.

## Project Structure

```
urdle/
├── index.html          # Main HTML page
├── css/
│   └── style.css       # Design system & component styles
├── js/
│   ├── words.js        # Word list
│   ├── game.js         # Game logic & key listeners
│   └── daily.js        # Daily word selection
├── assets/
│   └── icons/          # Icon assets
├── vercel.json         # Vercel deployment config
└── README.md
```

## Development

Open `index.html` directly or serve locally:

```bash
npx -y serve .
```

## Deployment

Push to GitHub and connect to [Vercel](https://vercel.com) for automatic deployments.
